# Простой сайт на PHP Laravel

## Описание
Скачать проект:  
```
git clone https://github.com/Processori7/siplelaravelsite.git
```
Перейти в директорию:  

```
cd siplelaravelsite
```
Установить composer: https://getcomposer.org/  
Выбрать php.exe  

Запустить сервер:  

```
php artisan serve.txt
```
